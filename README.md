# Acherna
마인크래프트 도구 모듈화 모드

Minecraft tool modularization mod

## 개요 / Overview
마인크래프트의 도구를 마음대로 커스터마이징 할 수 있게 해 주는 모드입니다.

재미있게 플레이하던 동일한 컨셉의 모드가 1.20.1 이후로 업데이트가 중단되어, 제가 새로 만들기로 했습니다.

his mod allows you to freely customize tools and equipment in Minecraft.
A mod with a similar concept that I thoroughly enjoyed playing stopped updating after version 1.20.1, which inspired me to create this brand-new project myself.

## 상세 / Detail
도구들은 총 7가지 분류를 가집니다
- 근접 무기
- 원거리 무기
- 도구
- 투구
- 흉갑
- 레깅스
- 부츠

도구는 모드에서 추가하는 특수한 작업대에서 상호작용하여 부품을 교체하거나, 수리하거나, 강화할 수 있습니다

Tools and gear are categorized into 7 distinct types:
- Melee Weapons 
- Ranged Weapons 
- Tools 
- Helmets 
- Chestplates
- Leggings
- Boots

Using a specialized workbench introduced by this mod, players can interactively swap out individual parts, repair their gear, or apply powerful upgrades.
